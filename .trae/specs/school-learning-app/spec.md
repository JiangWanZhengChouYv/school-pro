# 小学学习激励网页应用 - 产品需求文档

## Overview
- **Summary**: 一款面向小学教育场景的本地化学习激励网页应用，采用Vite+Vue3技术栈构建，支持题目管理、学生答题、AI出题与错题分析等核心功能。所有数据本地存储，打包后可直接双击运行，无需额外服务器环境。
- **Purpose**: 帮助家长为孩子创建个性化练习题，通过AI智能出题和错题分析提升学习效率，同时以游戏化的激励机制培养孩子的学习兴趣。
- **Target Users**: 
  - 小学生（6-12岁）：主要使用者，进行答题学习
  - 家长/老师：管理者，负责题目导入、配置和查看学习数据

## Goals
- 提供简单易用的题目管理系统，让非技术背景的家长也能轻松操作
- 构建友好的学生答题界面，符合小学生认知特点
- 集成AI能力，实现智能出题和错题分析
- 确保数据安全存储，所有数据本地化
- 打包后可直接双击运行，零部署成本

## Non-Goals (Out of Scope)
- 不支持多用户账号系统（单设备单用户使用）
- 不支持云端同步功能（纯本地应用）
- 不支持实时多人协作
- 不包含在线支付或订阅功能
- 不提供移动端原生App（仅网页应用）

## Background & Context
- 技术栈：Vite + Vue 3 + JavaScript
- 数据存储：localStorage（配置、密码、统计数据）+ IndexedDB（题库、图片）
- 加密算法：bcrypt.js（家长密码加密）
- AI服务：支持OpenAI兼容API和Anthropic API
- 构建方式：静态文件打包，通过file://协议直接打开
- UI风格：色彩鲜明、卡通化、交互简单，适合小学生使用

## Functional Requirements

### FR-1: 家长模式与权限控制
- 网页启动默认进入学生模式
- 右上角设有"进入家长模式"按钮
- 点击后弹出密码输入框，验证通过后切换至家长管理界面
- 首次进入家长模式需设置家长密码
- 家长密码使用bcrypt不可逆加密后存储在localStorage
- 所有题目编辑、类型管理、AI配置等操作均需家长权限验证
- 家长模式下可随时切换回学生模式

### FR-2: 题目类型管理
- 支持创建、查看、删除题目类型
- 类型以表格形式展示，包含类型名称、题目数量等信息
- 创建/编辑题目时可通过下拉选择已有类型或创建新类型
- 类型数据持久化存储

### FR-3: 题目导入与管理
- 支持手动录入题目，包含题干、答案、类型三个核心字段
- 题干支持富文本编辑（加粗、斜体、图片、换行等）
- 答案支持多种格式：文本、图片、多选选项等
- 题目列表支持按类型筛选、搜索、分页
- 支持编辑和删除已有题目
- 题目数据存储在IndexedDB中
- 支持题目图片上传，兼容JPG、PNG等常见格式

### FR-4: 学生答题系统
- 学生模式下展示做题界面
- 支持按题目类型选择练习
- 清晰的题目展示，支持图片显示
- 答题交互友好，支持输入框、选择题等多种形式
- 提交答案后即时反馈对错
- 自动记录每道题的答题结果

### FR-5: 答题记录与统计
- 自动记录每次答题的结果（正确/错误）
- 按题目类型、日期等维度统计正确率
- 统计数据结构化存储在localStorage中
- 学生模式下展示个人成绩和正确率
- 支持查看历史错题集

### FR-6: AI服务配置
- 提供AI服务API配置界面
- 支持OpenAI兼容API模式配置（API密钥、Base URL、模型名称）
- 支持Anthropic API模式配置（API密钥、模型名称）
- 两种模式可切换，配置数据加密存储在localStorage
- 提供连接测试功能验证配置有效性

### FR-7: AI智能出题
- 支持用户输入出题要求（如"小学三年级数学乘法题"）
- 设计系统提示词模板，指导AI按指定格式输出题目
- 系统提示包含现有题目类型列表
- 允许AI根据内容创建新类型
- 实现tool-call/function calling机制，确保AI输出结构化数据
- 生成的题目可预览、编辑后一键保存到题库

### FR-8: AI错题分析
- 支持选择学生错题集进行分析
- 设计系统提示词，指导AI分析题目难度、知识点、错误原因
- 实现tool-call机制，使AI输出结构化分析结果
- AI对学生答案进行评分并提供改进建议
- 分析结果可保存，供后续复习参考

## Non-Functional Requirements

### NFR-1: 可用性
- 界面设计符合小学生认知特点，色彩鲜明、交互简单
- 家长模式操作直观，适合非技术背景用户
- 响应式设计，适配桌面和平板设备
- 关键操作有明确的视觉反馈和提示

### NFR-2: 性能
- 题目列表加载时间 < 1秒（1000题以内）
- AI出题响应时间 < 30秒（取决于网络）
- 页面切换流畅，无明显卡顿

### NFR-3: 安全性
- 家长密码使用bcrypt加密存储，不可逆
- API密钥等敏感信息加密存储
- 所有数据仅存储在本地，不上传任何服务器
- XSS防护，用户输入内容安全处理

### NFR-4: 可维护性
- 代码结构清晰，模块化设计
- 关键代码有完整注释
- 组件复用性高
- 错误处理完善，有友好的用户提示

### NFR-5: 可移植性
- 打包后为纯静态文件，可通过file://协议直接运行
- 不依赖任何后端服务
- 兼容主流浏览器（Chrome、Edge、Safari、Firefox）

## Constraints
- **技术**: 必须使用Vite + Vue 3技术栈
- **部署**: 必须支持file://协议直接打开，无需服务器
- **存储**: 受浏览器存储容量限制（IndexedDB通常50MB-1GB）
- **AI服务**: 需用户自行提供API密钥，应用本身不提供AI服务
- **浏览器**: 需支持IndexedDB和ES6+的现代浏览器

## Assumptions
- 用户使用现代浏览器（Chrome 90+、Edge 90+、Safari 14+、Firefox 88+）
- 用户设备有网络连接（使用AI功能时需要）
- 单设备单用户使用，不需要多账号切换
- 题目数量在合理范围内（10000题以内）
- 用户具备基本的电脑操作能力

## Acceptance Criteria

### AC-1: 家长密码设置与验证
- **Given**: 用户首次进入家长模式
- **When**: 用户设置家长密码并确认
- **Then**: 密码以bcrypt加密形式存储在localStorage，下次进入需输入正确密码才能访问家长模式
- **Verification**: `programmatic`
- **Notes**: 密码强度至少6位，两次输入一致

### AC-2: 题目类型管理
- **Given**: 用户处于家长模式
- **When**: 用户创建新的题目类型
- **Then**: 类型列表中显示新类型，创建题目时可在下拉框中选择该类型
- **Verification**: `programmatic`

### AC-3: 题目创建与存储
- **Given**: 用户处于家长模式且已创建至少一个题目类型
- **When**: 用户填写题干、答案，选择类型后保存题目
- **Then**: 题目成功存储在IndexedDB中，在题目列表中可查看
- **Verification**: `programmatic`
- **Notes**: 题干支持富文本和图片

### AC-4: 学生答题与结果记录
- **Given**: 系统中已有可用题目，用户处于学生模式
- **When**: 学生选择题目类型并完成答题
- **Then**: 系统即时反馈对错，并将答题记录保存，正确率统计更新
- **Verification**: `programmatic`

### AC-5: AI服务配置
- **Given**: 用户处于家长模式
- **When**: 用户选择API类型并填入正确的API密钥等信息
- **Then**: 配置保存成功，连接测试返回成功
- **Verification**: `programmatic`
- **Notes**: 支持OpenAI兼容和Anthropic两种模式

### AC-6: AI智能出题
- **Given**: AI服务配置正确，用户处于家长模式
- **When**: 用户输入出题要求并点击生成
- **Then**: AI返回结构化题目数据（题干、答案、类型），用户可预览并保存到题库
- **Verification**: `programmatic`
- **Notes**: 使用tool-call机制确保输出格式正确

### AC-7: AI错题分析
- **Given**: AI服务配置正确且存在错题记录
- **When**: 用户选择错题集并请求分析
- **Then**: AI返回结构化分析结果，包含难度、知识点、错误原因、改进建议
- **Verification**: `programmatic`
- **Notes**: 使用tool-call机制确保输出格式正确

### AC-8: 模式切换
- **Given**: 用户处于学生模式
- **When**: 点击右上角"进入家长模式"按钮并输入正确密码
- **Then**: 页面切换到家长管理界面
- **Verification**: `programmatic`

### AC-9: 打包后可直接运行
- **Given**: 应用已完成构建打包
- **When**: 用户双击index.html文件
- **Then**: 应用在浏览器中正常运行，所有功能可用
- **Verification**: `human-judgment`
- **Notes**: 需确保静态资源路径正确，支持file://协议

### AC-10: UI/UX符合小学生特点
- **Given**: 学生模式答题界面
- **When**: 小学生使用应用答题
- **Then**: 界面色彩鲜明、字体适中、操作简单，交互有动画反馈
- **Verification**: `human-judgment`
